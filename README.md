# python-project-template

## Feature
- poetry
- flake8
- black
- isort
- pytest


## How to use
```
# set up virtual env
poetry shell

# install packages
poetry add package-name

# run
python -m python_project_template

# lint
poetry run flake8

# test
poetry run pytest
```

## MEMO
```
# create new project
poetry new project-name
```
